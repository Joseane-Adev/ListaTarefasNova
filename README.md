Programa simples em python usando classes. Permite adicionar, listar tarefas e remover.

🛠 Funcionalidades
Adicionar tarefa → O usuário digita uma nova tarefa e ela é registrada.

Listar tarefas → Mostra todas as tarefas numeradas.

Remover tarefa → Remove uma tarefa escolhida pelo número.

Sair → Encerra o programa.
